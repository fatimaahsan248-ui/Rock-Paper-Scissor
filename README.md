# Rock-Paper-Scissor
A simple Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer, which makes random choices. The game displays both players' choices, announces the winner of each round, and keeps track of the score.
# Rock Paper Scissors

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

- Play against the computer
- Random computer moves
- Live score tracking
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
